# Presence Synchronization Engine

Presence Synchronization Engine is a real-time chat application backend built using Node.js and Socket.IO.  
It keeps user presence, messages, typing indicators, and message status synchronized across multiple users in real time.

---

## Features

- Group chat
- Private one-to-one chat
- Online and offline user status
- Typing indicators
- Message sent, delivered, and seen status
- Real-time updates using WebSockets

---

## Tech Stack

- Node.js
- Express.js
- Socket.IO
- HTML, CSS, JavaScript
